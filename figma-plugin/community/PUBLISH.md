## description\*

```
Open Icons puts 17,000+ free, open-source SVG icons from 14 of the best packs — Lucide,
Phosphor, Tabler, Material Symbols, Heroicons, Feather, Bootstrap, Remix, Eva, Pixelart,
Flagpack, Iconoir, Flowbite and Doodle — one search away, without leaving Figma.

What you can do:
• Search every pack at once, or browse a single library.
• Insert any icon as a clean, editable vector — pick the size, lock the aspect ratio,
  clip the frame, all set up for you.
• Switch variants (outline / solid / fill, weights, duotone) per pack.
• Identify any selected icon — the plugin tells you which library it's from, by its
  layer name or even by its shape.
• Swap packs in one click: turn a Lucide icon into Phosphor, Tabler, Material… in place.
• Select many icons from different libraries and swap them ALL to one pack at once —
  exact name match first, then the closest match by shape.
• Save favorites, reuse recents, and switch the UI between light, dark and your device theme.

Nothing is bundled or out of date: every icon is fetched live from its source pack, so
you always get the latest release. All icons keep their original open-source licenses.

🌐 Website: https://open-icons.vercel.app — browse, copy SVGs, grab CDN links, and read
the docs in your browser too.

———

Made by rakibulism. If Open Icons saves you time, follow along and say hi on X:
https://x.com/rakibulism
```

## Category & subcategory (one each)

**Category:** `design tools`
**Subcategory:** `Import & export`

*Rationale:* the plugin's core action is bringing icons **into** a Figma file (and out as SVG) — "Import & export" is the most accurate functional match, and `design tools` is where designers look for icon-insertion utilities. If Figma lets you pick a standalone category with no subcategory, **`Icon Packs`** is the most literal match — use that instead.
